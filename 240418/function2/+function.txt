# function 여러가지 함수 

1. 복습)함수는 기존 PHP에서 제공하는 내장함수가 있다.

2. 복습)또한 유저 정의 함수로 직접 만들어 사용 할 수 있는 함수가 있다.
  ex)  function정의=>어떤 처리를 할 것인가.(){};

3. 문자열과 수치 처리 함수
3-1. strlen(), mb_strlen(), substr(), str_replace() 등등 
3-2. nl2br() 텍스트 줄바꿈 <br>
3-3. isset() 변수, 배열 요소 체크 함수 / 있으면 true 없으면 false
3-4. empty() 변수나 배열이 비어있을때 체크 함수 / 비어있으면 1, 값 있으면 0
3-5. trim() 문자열 전, 후 공백 삭제 함수 
3-6. fopen(파일명, 오픈모드(r,w,a)) 파일 불러오는 함수 
3-7. fclose() 파일 닫기 / 파일에 대한 변경 저장 
3-8. fgets() 1행씩 읽어들임, feof()랑 많이 쓰임 -> 파일의 끝인지 아닌지 확인하는 함수
3-9. fputs() 파일안에 쓰는 함수 
3-10. file_get_contents() 파일 열고 내용 반환
3-11. file_put_contents() 파일 만들고 내용 반환
