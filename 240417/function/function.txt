# function 함수

1. 함수는 편하게 만들어놓은 미니프로그램이라 생각하면 됨
2. 직접 만드는 유저함수가 있고 내장함수가 존재함
3. 내장함수의 종류는 여러가지로 php 메뉴얼 보면서 그때그때 찾으면서 해도 괜찮음
4. 설명보고 모르겠으면 다른 사람 코드 혹은 예시 보면서 활용
5. 함수는 제어문 활용을 더욱 더 편하게 만듬

# requrie
1. 함수를 파일로 만들어서 불러오는 것도 가능
2. 그럴때는 require를 사용해서 불러오면 됨 ex) require 'test.php'; 
3. 코드와 늘어나 가독성이 떨어지거나 복잡함을 해결할 수 있음
4. 대신 함수 파일 관리 및 경로지정을 잘 해줘야함

#include
1. include , include_once 두 종류가 있음
2. include 요청할떄마다 파일 불러옴 
3. include_once = 처음 1회만 파일을 불러고 이후는 스킵
